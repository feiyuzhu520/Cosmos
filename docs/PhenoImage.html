Steps:
1. Load files
2. Select the ROI
3. Apply the filter
4. Select functions(sample functions are included in the installation file)
5. Generate code for HCC
6. Execute for all images
7. Save results

Path should be a regular expression for the image files. * can be used to represent characters (e.g. C:\*\Vis_SV_*\*.png).

Slider can be used to select the image that is shown.

Filter operation should be a logical operation for RGB or HSV channels (e.g. g<150 & g>r+20 & g>50 )

Instructions for Job Submission on HCC

1. Save the generated code (e.g. as imageHCC.m);The program will generate a text file containing all the image names (e.g. ls *maize*/Vis_SV*/*.png > filenames.txt);
2. Use the included .slurm file to submit all the jobs on HCC.
